The first command if inititating a project is git init. This initialises it as a git repository.

This repo is empty. So we tell git that we have new files (untracked) that need to be tracked. Git will add these in the next commit. We Stage the files to be tracked. git add .

This adds all the files or you could name specific ones.

git add is powerful. You can git add *.html, (just html files) or *.php etc..
